Vertical navigation with jQuery

### Usage instructions

To instantiate the plugin use any selector:

```javascript
$('#leftNavigation').ssdVerticalNavigation();
```

There are also a few options you can use:

```javascript
$('#leftNavigation').ssdVerticalNavigation({
    classMaster : 'master',
    classActive : 'active'
});
```
